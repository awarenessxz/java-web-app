# API Gateway

In general, an API Gateway is part of an API Management System. It can have the following features:
- Reverse Proxy
- Service Discovery
- Authentication
- Loading Balancing
- Rate Limiting (prevent overuse / abuse)
- Analytics and Monitoring
- Caching

## Folder Structure

is about.
